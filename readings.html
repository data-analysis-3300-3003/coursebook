Course literature

Context readings

Each week you will be provided with a short “context” reading. These context readings cover important current topics, debates, and challenges with the use of geospatial data and geospatial / agricultural data science tools. These readings will build your awareness of how geospatial data and the data science skills you are learning through this course are applied.

You will be asked questions from these readings in each week’s quiz.

Week 1

Ethics in geo

Big geospatial datasets, artificial intelligence, and advances geospatial applications and software are raising new and challenging questions for the ethical use of these technologies.

  • Ethics in geo (Tom McWright)
  • Locus charter

Week 2

Open Geospatial Consortium - Standards

A standard is a specification of rules and guidelines that can be used to define software interfaces or encoding of data. This week’s readings introduce the Open Geospatial Consortium standards and encourage you to think about why standards are important and the distinction between standards and file formats.

  • Interoperability
  • Understanding OGC standards
  • The need for OGC standards (Please watch the video too)
  • Data encoding standards
  • GeoPackage (Read the paragraphs of the landing page for the GeoPackage website. In particular, look out for the distinction between the GeoPackage format and the GeoPackage Encoding Standard)

Week 3

Introduction to machine learning

Machine learning is increasingly being used to extract insights from geospatial datasets. This video by Climate Change AI provides an introduction to machine learning concepts and applications of machine learning in responding to climate change.

  • Introduction to machine learning (Climate Change AI)

Week 4

Satellite data, machine learning, and sustainable development

Read Using satellite imagery to understand and promote sustainable development by Burke et al. (2021) which discusses current state-of-the-art, opportunities, and challenges for using satellite data with machine learning for sustainable development. In particular, focus your reading on the following sections: Introductory paragraps, The availability and reliability of data, and Modeling approaches using satellite imagery to predict sustainability outcomes.

  • Using satellite imagery to understand and promote sustainable development

Week 5

Artificial intelligence and agriculture

Read the Microsoft blog about project FarmVibes.AI which demonstrates how various geospatial datasets and machine learning technologies can be applied to support farm management.

  • Microsoft open sources its ‘farm of the future’ toolkit

Week 6

FAIR data

FAIR is a set of principles to improve the Findability, Accessibility, Interoperability, and Reuse of data. Ensuring that data adheres to these principles makes it more relevant and easy to use in applications and for decision making.

  • FAIR principles
  • FAIR FAQs
  • FAIR academic paper (Note: this is extra reading)

Week 7

Cloud native geospatial data
